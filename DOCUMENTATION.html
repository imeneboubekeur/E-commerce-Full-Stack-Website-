Technical Documentation, Database Schema & System Architecture
React 19 & SSR Node.js / Express PostgreSQL DB Stripe Payments Cloudinary Storage

Executive Summary

This document provides a comprehensive technical overview of the Furniture E-Commerce application. It covers full end-to-end system architecture, workflow flowcharts for customers, administrators, and payment webhooks, database entity-relationship (ER) diagrams, detailed PostgreSQL schema definitions, and complete REST API specifications.

Key System Specifications

  • Frontend Architecture: React 19, Redux Toolkit, React Router 7, Webpack Server-Side Rendering (SSR)
  • Backend REST API: Express.js with JSON Web Tokens (JWT) & bcrypt authorization
  • Database Engine: PostgreSQL with foreign key relational integrity & JSONB dynamic configurations
  • Third-Party Integrations: Stripe Checkout & Webhooks, Cloudinary Media Assets Management

4. Detailed Database Schema Specification

Below is the detailed PostgreSQL database table specification for all 9 tables initialized via backend/database/initDB.js.

4.1 Table: users

Column Data Type Constraints & Defaults Description
idSERIALPRIMARY KEYUnique user identification key
emailVARCHAR(255)NOT NULL, UNIQUEUser login email address
passwordVARCHAR(255)NOT NULLBcrypt hashed password
nameVARCHAR(255)NOT NULLFull user name
phoneVARCHAR(20)NULLContact phone number
addressTEXTNULLDefault delivery address
image_urlTEXTNULLCloudinary avatar URL
image_public_idVARCHAR(255)NULLCloudinary asset reference ID
reset_tokenTEXTNULLPassword reset verification token
reset_token_expiryTIMESTAMPNULLReset token expiration time
created_atTIMESTAMPDEFAULT CURRENT_TIMESTAMPRecord creation timestamp

4.2 Table: categories

Column Data Type Constraints & Defaults Description
idSERIALPRIMARY KEYUnique category identifier
nameVARCHAR(100)NOT NULLCategory name (e.g. Living Room)
image_urlTEXTNULLCategory showcase image URL
image_public_idVARCHAR(255)NULLCloudinary asset ID

4.3 Table: products

Column Data Type Constraints & Defaults Description
idSERIALPRIMARY KEYUnique furniture product ID
nameVARCHAR(255)NOT NULLItem product title
descriptionTEXTNULLDetailed furniture description
priceDECIMAL(10,2)NOT NULLItem price in USD
category_idINTREFERENCES categories(id)Foreign key to parent category
stockINTEGERDEFAULT 0Available stock inventory count
skuVARCHAR(100)UNIQUEStock Keeping Unit barcode
image_urlTEXTNULLPrimary product image
